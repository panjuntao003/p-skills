# Invest

This is a human-readable investment research folder maintained by `p-invest`.

Start here:

- `0-dashboard/now.md` - current view
- `2-companies/` - company research
- `3-decisions/trades.md` - trade log
- `5-reviews/monthly/` - monthly reviews
- `6-system/rules/` - investment rules

You can open this folder directly in Obsidian.
