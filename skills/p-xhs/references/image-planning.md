# Stage 5：配图方案

## 图片规范

### 尺寸
| 类型 | 尺寸 | 比例 |
|------|------|------|
| 封面 | 1080×1440 px | 3:4 |
| 内图 | 1080×1440 px | 3:4 |

### 数量
| 笔记类型 | 推荐张数 |
|---------|---------|
| 标准教程/评测 | 6-9 张（含封面） |
| 简单分享/观点 | 3-5 张（含封面） |
| 纯文字观点 | 1 张（封面） |

### 风格

侦探奎因账号风格公式：
```
纯色背景 + 大标题 + 真实截图
```

- **封面**：纯色背景 + 大标题（≤7 字）+ 产品/AI 工具截图
- **内图**：步骤截图/代码片段/对比图/数据图表 + 文字说明
- **不要**：过度设计的拼贴、花哨边框、多图拼接

## 工作流程

### Step 1：分析内容
从文案提取：内容类型、核心素材、视觉钩子

### Step 2：封面方案
- 背景色（从账号已有风格延续）
- 大标题 ≤7 字
- 核心图描述
- 可选小标签（"真实案例""实测""对比"）

### Step 3：内图方案
按阅读顺序规划：

| 图号 | 内容 | 说明 |
|------|------|------|
| 图2 | 承接封面 | 展开核心信息 |
| 图3 | 步骤/数据/对比 | — |
| 末图 | 总结/金句 | 留信息让人截图保存 |

### Step 4：输出完整方案

```markdown
## 配图方案

### 封面
- 设计：背景色 + 大标题 + 核心图描述
- 输出规范：1080×1440, 3:4

### 内图（共 X 张）
| 图号 | 内容 | 说明 |
|------|------|------|
```

### 文件命名
```
封面：YYYY-MM-DD-标题关键词-封面.png
内图：YYYY-MM-DD-标题关键词-内图1.png
```
